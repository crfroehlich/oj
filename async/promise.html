ajaxPromise

OJ.async.ajaxPromise converts an AJAX XmlHttpRequest into a Promise. See also Promise.resolve.

OJ.async.register 'ajaxPromise', (ajax) -> promise = Promise.resolve ajax promise.abort = ajax.abort promise.readyState = ajax.readyState promise

all

OJ.async.all takes an array of functions and returns a promise representing the success of all methods or the failure of any method. See also Promise.all.

OJ.async.register 'all', (initArray) -> reqs = initArray or [] promise = Promise.all(reqs) promise.push = (item) -> reqs.push item return promise
